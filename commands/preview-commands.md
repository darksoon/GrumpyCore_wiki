# Preview-Commands

Mit `/preview` können Admins Nachrichten testen ohne dass jemand joinen/leavt muss.

**Berechtigung:** Server verwalten

---

| Command | Funktion |
|---|---|
| `/preview welcome` | Welcome-Banner mit eigenem Avatar + aktueller Memberzahl |
| `/preview welcome @user` | Welcome-Banner als ob dieser User gejoint wäre |
| `/preview leave` | Alle konfigurierten Leave-Nachrichten als Liste |
| `/preview verify` | Verify-Embed mit deaktiviertem Button (nur Vorschau) |

Alle Previews sind **nur für dich sichtbar** (ephemeral).
