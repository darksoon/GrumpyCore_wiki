# Ticket-Commands

| Command | Berechtigung | Funktion |
|---|---|---|
| `/ticket close [grund]` | Support-Rolle oder Ticket-Ersteller | Ticket schließen |
| `/ticket claim` | Support-Rolle | Ticket claimen (Zuständigkeit übernehmen) |
| `/ticket add @user` | Support-Rolle | User zum Ticket-Thread hinzufügen |
| `/ticket remove @user` | Support-Rolle | User aus Ticket-Thread entfernen |
| `/ticket stats` | Support-Rolle | Ticket-Statistiken anzeigen |

---
