## 🚀 Schnellstart

1. Dateien auf den Pelican-Server hochladen
2. `configs/config.yml` ausfüllen (Token, Channel-IDs, Rollen-IDs)
3. Bot starten → Konsole zeigt welche IDs noch fehlen
4. Message-IDs aus der Konsole kopieren und eintragen
5. `reload` in der Pelican-Shell
